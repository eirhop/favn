# Historical Refactor Material

This directory records completed migration plans, review packets, and phase TODOs.
It is retained for design history, not as current architecture or backlog.

PostgreSQL Storage V2 removed the old generic storage adapter, memory backend, and
SQLite backend. Current implementation guidance lives under `docs/storage/`,
`docs/structure/`, and `docs/production/`; current forward work lives only in
`docs/ROADMAP.md` and GitHub issues.
