# Documentation Archive

This directory preserves historical Favn planning and AI-agent reference documents.

Archived documents are not current public product documentation. Treat them as
implementation history, migration context, or prior review material. Prefer the
current docs under `docs/architecture/`, `docs/adapters/`, `docs/operators/`,
`docs/production/`, `docs/structure/`, and `apps/favn/guides/` when writing new
user, operator, or contributor documentation.

## Sections

- `ai-planning/`: top-level issue plans, broad implementation plans, PR notes,
  tasklists, and architecture/scope references that were primarily used to guide
  AI-agent coding work.

Do not delete archived documents casually. Move additional historical docs here
only when a current index or replacement doc makes their status clear.
