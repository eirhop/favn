# AI Planning Archive

These documents are historical AI-agent planning and implementation references.
They may describe designs that were superseded, partially implemented, or folded
into current product docs.

Use current docs first:

- `docs/DOCUMENTATION_GUIDE.md` for documentation standards.
- `docs/DOCUMENTATION_PLAN.md` for the current documentation initiative.
- `docs/FEATURES.md` for implemented capabilities.
- `docs/ROADMAP.md` for forward-looking product direction.
- `docs/architecture/`, `docs/adapters/`, `docs/operators/`, and
  `docs/production/` for current explanation, reference, and operator docs.
- `docs/structure/` for app and path ownership maps.

Archived files currently include:

- Issue implementation plans: `ISSUE_*.md`.
- Broad implementation plans: `*_PLAN.md`.
- Historical PR notes: `*_PR.md`.
- Historical tasklists: `*_TASKLIST.md`.
- Historical architecture and scope references: `*_ARCHITECTURE.md`,
  `sql_adapter_scope.md`, `REFACTOR.md`, and `refactor_review_standard.md`.

If a current doc links here, the link should make clear that the target is
historical context rather than supported product behavior.
