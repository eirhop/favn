# Point-in-Time Reports

Every file in this directory is historical review evidence. Reports describe the
commit and architecture available when the review was written; findings may be
fixed, superseded, or invalid after PostgreSQL Storage V2.

Use current code plus `docs/FEATURES.md`, `docs/ROADMAP.md`, and
`docs/production/README.md` for present status. Do not open implementation work
from a report without reproducing the finding against current `main`.
